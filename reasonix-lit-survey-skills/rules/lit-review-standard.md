# 文献调研深度标准

## 每篇论文必须回答四个问题
1. **做了什么** — 一句话概括核心工作
2. **创新点在哪** — 为什么能发这个级别的期刊？方法创新？性能突破？机制揭示？
3. **关键性能指标** — 具体数字，需要标红
4. **对我的启发** — 与个人研究体系的关联，可借鉴的思路/方法/机制

## 输出要求
- 不能只看摘要，必须阅读正文（至少Intro+Results+Conclusion）
- 关键性能数字用红色标注
- 每篇论文的"对我的启发"要具体，不能泛泛而谈
- 报告要有深度，不是罗列摘要

## 顶级期刊发表逻辑分析
- Nature/Science: 解决领域公认的核心难题，开辟新方向
- Nature子刊: 在某一方向上有重大突破，性能/机制/方法上的飞跃
- JACS/Angew: 分子设计/合成策略的创新性+性能突破
- AM/AFM: 材料设计+性能+应用验证的完整故事
