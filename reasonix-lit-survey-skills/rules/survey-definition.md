# 调研 = 文献调研（完整定义 + 铁律）

## 调研的四步铁律（不可跳过任何一步！）

```
① 搜索 → ② 筛选 → ③ 下载 → ④ 汇总
```

**每步都不可跳过。尤其是第③步"下载"，必须把筛选出的论文全部下载下来！**

## 完整流程（7步）

1. **理解用户体系**：先读用户论文/数据，搞清楚材料、工艺、参数、卡在哪
2. **诊断根因**：对比文献参数 vs 用户参数，找出差距
3. **搜索文献** ⬅️ 铁律第①步：scholar-search + web_search，只搜白名单期刊
4. **筛选** ⬅️ 铁律第②步：去重、排除无关、优先有工艺参数的
5. **下载全部** ⬅️ 铁律第③步：**不询问用户，全部直接下载！** Wiley/ACS → MCP 浏览器 CDP 9223；OA → curl。**论文 PDF 必须落地到 papers/ 目录**
6. **对比分析 + 提出方案**：用户参数 vs 文献参数对比表，按优先级排列方案
7. **汇总输出** ⬅️ 铁律第④步：Word 报告 + PDF 全部到位

## ⚠️ 严禁行为

- ❌ 只诊断不下载（"我读了你已有的论文就够了"——不够！必须搜新论文并下载）
- ❌ 下载前问用户"要下载哪些"
- ❌ 搜索非白名单期刊
- ❌ 只下载部分论文
- ❌ 不写 Word 报告

## 期刊白名单

Nature / Science / Nature Communications / Nature Photonics / Nature Materials / Nature Chemistry / Nature Physics / Nature Nanotechnology / Science Advances / Angewandte Chemie / JACS / Advanced Materials / Advanced Energy Materials / Advanced Functional Materials（AFM 最低级别）

**Why**: 2025-05-30 某次调研中，只做了诊断但没有搜索和下载新论文。用户明确指出：搜索→筛选→下载→汇总这个流程不能忘，调研一定要下载文献。

**How to apply**: 用户说"调研XX"→ 先理解体系 → 诊断根因 → **必须搜索新论文** → **必须下载全部** → 汇总报告。缺一步都不叫调研。
